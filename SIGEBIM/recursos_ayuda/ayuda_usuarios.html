Manual Técnico: Gestión de Usuarios

Módulo: Administración de Lectores y Accesos | Versión: 1.0

1. Descripción General

Este formulario controla quién puede ingresar al sistema y llevarse material. Gestiona la creación de perfiles, asignación de credenciales (Username/Password) y niveles de autorización (Roles).

2. Detalle de Campos del Formulario

CampoRegla Técnica de Uso
DNI8 dígitos exactos. Sirve como identificador principal para préstamos.
Nombres y ApellidosBloqueados. El sistema los obtiene automáticamente de la base de datos de identidad al validar el DNI.
UsernameNombre de usuario único para inicio de sesión (Ej: rmartinez). No usar espacios.
PasswordContraseña del sistema. Mantener la confidencialidad.
Rol de Usuario'Administrador' (acceso a config/reportes) o 'Usuario' (solo catálogo/préstamos).
Teléfono / CorreoDatos de contacto para el envío de alertas de vencimiento de devoluciones.

3. Listado de Acciones: Lo que SE DEBE hacer

  • Validar siempre la identidad: Digite el DNI y presione el botón de validación. Verifique que los nombres devueltos correspondan al lector físico frente a usted.
  • Búsqueda Multicriterio: Utilice la barra de búsqueda superior ingresando el Username para verificar accesos, o el Apellido si el usuario olvidó su número de documento.
  • Actualización segura: Para cambiar una contraseña o rol, busque al usuario, modifique el campo correspondiente y guarde. El DNI permanecerá inmutable.
  • Uso de la Escoba: Utilice el ícono de escoba para vaciar el formulario completamente antes de atender a un nuevo usuario en la fila.

4. Restricciones: Lo que NO SE DEBE hacer

  • No intente forzar los campos de Nombre: Están bloqueados por arquitectura del sistema para evitar suplantaciones de identidad.
  • No asigne el rol 'Administrador' indiscriminadamente: Esto otorga acceso a parámetros sensibles como precios de multas y rutas de bases de datos.
  • No duplique Usernames: El sistema validará que cada nombre de usuario sea único.

🛠️ Solución de Problemas Frecuentes

Error: "DNI no encontrado o error de validación".
Solución: Verifique su conexión a la red o asegúrese de que ha digitado exactamente 8 números sin letras.
