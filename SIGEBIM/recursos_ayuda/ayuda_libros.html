Manual Técnico: Gestión de Libros

Módulo: Catalogación e Inventario | Versión: 1.0

1. Descripción General del Módulo

Este módulo es el núcleo del sistema SIGEBIM. Permite a los bibliotecarios registrar, modificar, consultar y eliminar material bibliográfico. Soporta tanto formatos físicos como recursos digitales mediante enlaces directos (URL).

2. Reglas de Llenado de Campos (Diccionario de Datos)

CampoDescripción y Regla de Uso
ISBNCódigo internacional de 13 dígitos. No admite letras ni guiones. Es la llave principal.
Título / Autor / EditorialCampos de texto obligatorios. Deben ingresarse respetando mayúsculas y minúsculas.
Categoría / IdiomaListas desplegables. Seleccione la opción que mejor describa la obra para facilitar el filtrado.
Tipo (Físico/Digital)Si es "Físico", el campo URL se bloquea. Si es "Digital", se habilita la URL obligatoriamente.
URLEnlace web para descarga o lectura. Formato válido requerido (ej. https://...).
EstadoAutomatizado. No requiere intervención manual. Se asigna como 'Disponible' al crear.

3. Listado de Acciones: Lo que SE DEBE hacer

  • Limpiar antes de registrar: Utilice siempre el botón de la Escoba antes de registrar un nuevo libro para asegurar que no queden datos ocultos de una consulta anterior en la memoria del formulario.
  • Actualizar correctamente: Para modificar, primero realice una búsqueda, seleccione la fila del libro en la tabla, cambie el dato erróneo y presione 'Actualizar'.
  • Uso de Búsqueda Específica: Para encontrar un ejemplar exacto, escriba el ISBN completo o el Autor exacto en la barra de búsqueda y presione Enter.
  • Uso de Búsqueda General: Si un usuario pide "Libros de Ciencia Ficción", deje la barra de texto vacía y seleccione la categoría en el menú desplegable para filtrar toda la tabla.

4. Restricciones: Lo que NO SE DEBE hacer

  • No intente registrar ISBN duplicados: El sistema rechazará la operación y mostrará una alerta de error de base de datos. Cada libro físico debe tener un registro único.
  • No borre libros con préstamos activos: El sistema prohíbe eliminar un libro si su estado actual es 'No Disponible'. Primero debe registrarse la devolución.
  • No ingrese URLs inválidas: En libros digitales, no escriba texto común en la casilla de URL, debe ser un hipervínculo funcional.

🛠️ Solución de Problemas Frecuentes (Troubleshooting)

Error: "El campo URL está bloqueado pero el libro es digital".
Solución: Asegúrese de haber seleccionado "Digital" en el menú desplegable. El campo solo escucha el cambio de estado de esa lista.

Error: "La tabla no muestra todos los libros tras una búsqueda".
Solución: Presione el botón de la escoba para limpiar los filtros y presione buscar con el campo de texto vacío para recargar el inventario total.
